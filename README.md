# 🥋 JMD Taekwondo Club – Official Website

A fully responsive and content-rich website developed for JMD Taekwondo Club, a Delhi-based martial arts organization. The site highlights major achievements, introduces the coaching team, and showcases event galleries with a clean and intuitive layout.

## 🔧 Features

- 🏆 Structured Achievements Section with event-wise medal details
- 👨‍🏫 Coach Profile Cards with hover effects
- 🖼️ Image Gallery with responsive layout and zoom-on-hover animations
- 📱 Fully responsive design for mobile, tablet, and desktop
- 🧭 Smooth navigation and clean content structure

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Flexbox & Media Queries

## 📌 Status

✅ Completed (2024) – Delivered as a freelance project  
🛠️ Actively maintained with minor UI/UX refinements
